# README

## Endpoints:
`/api/send_message`: (POST) accepts json, "message"-tag is converted into a matrix message <br>
`/api/send_message_with_title`: (POST) accepts json, "message"-tag is converted into a matrix message with "title"-tag as title<br>
`/api/send_image_url`: (POST) accepts URL, whose image is downloaded and send as a matrix message<br>
`/api/send_collated_messages`: (GET) sends all collated messages<br>
